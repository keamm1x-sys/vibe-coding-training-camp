隐私政策

《Vibe Coding 新手训练营》是一款离线学习 App。本政策说明 App 如何处理数据和隐私。

我们收集哪些信息

第一版 App 不需要账号，不连接服务器，不收集个人数据，不进行广告追踪，也不上传学习记录。

学习进度

课程进度、测验结果和设置仅保存在你的设备本机，用于显示学习状态和继续学习。删除 App、清除本机数据或更换设备后，进度可能无法恢复。

网络与第三方服务

第一版课程、插图、测验和资料库均为本地资源，不依赖我们的服务器。App 不包含账号系统、远程数据库、推送、广告 SDK 或第三方统计 SDK。

设备权限

第一版 App 不主动申请通讯录、相册、定位、麦克风、摄像头等敏感权限。如未来版本新增相关能力，会在版本说明和隐私政策中说明用途。

儿童与未成年人

本 App 面向希望理解 AI 辅助开发流程的学习者，不专门面向儿童设计。未成年人购买或使用时，建议在监护人了解后进行。

政策更新

如果后续版本加入联网功能、账号、同步、反馈或统计能力，本政策会随版本更新同步调整，并在商店更新说明中提示。

联系我们

如需反馈隐私相关问题，请通过 GitHub Issues 联系：https://github.com/keamm1x-sys/vibe-coding-training-camp/issues。
